# IDACodeEditor
### Simple IDA Pro Code Editor

### Available 6.8 / 7

## Installation

### Modules
```
pip install tornado
```

### IDA Code Editor
```
IDACodeEditor folder and IDACodeEditor.py -> \IDA 6.8 or 7\plugins
```

### front-end
```
open index.html in browser
```
